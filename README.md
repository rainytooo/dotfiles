Dotfiles
========

这个项目就是保存我的osx常用配置的,这样在新机器或者新的系统环境下能够快速的构建原有的习惯的开发环境.

主要包含以下内容

* 安装初始化脚本
* vim配置和主题等
* tmux配置和主题等
* 安装一些必要的工具等等


Installation
------------

先确保安装xcode和相应的工具

```bash
xcode-select --install
```

然后拉取代码,并执行初始化脚本

``` bash
git clone git://github.com/rainytooo/dotfiles ~/.dotfiles
cd ~/.dotfiles
./install.sh
```
